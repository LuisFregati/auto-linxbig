🚀 Auto-LinxBIG: Automação de Suporte
Este projeto foi desenvolvido para otimizar o fluxo de trabalho no atendimento de suporte técnico do sistema ERP Linx BIG (Segmento Farma).  

📋 Sobre o Projeto
O script automatiza a geração de relatórios de incidentes, permitindo uma visualização clara e rápida dos chamados críticos. Ele simula a extração de dados (SQL) e transforma informações brutas em um relatório visual formatado.  

🛠️ Tecnologias Utilizadas
Python: Lógica de processamento de dados e automação.  

HTML/CSS: Estruturação e estilização do relatório final.  

SQL (Conceitos): Estruturação dos dados baseada em consultas de banco de dados ERP.  

💡 Funcionalidades
Processamento de lista de chamados por módulo (Fiscal, PBM, Financeiro, etc.).  

Destaque visual para chamados de Prioridade Alta.  

Geração automática de arquivo HTML pronto para leitura ou envio por e-mail.  

🔧 Como usar
Certifique-se de ter o Python instalado.  

Execute o arquivo main.py.

O arquivo relatorio_suporte.html será gerado automaticamente na pasta do projeto.
